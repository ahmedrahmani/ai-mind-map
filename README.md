# 🧠 AI Mind Map · Deep Hierarchical Encyclopedia

> Explore **2100+ AI topics** in a beautifully organized, interactive mind map.

From **historical foundations** to **cutting-edge agentic frameworks**, this project is your **ultimate one-stop knowledge tree for Artificial Intelligence**.

---

## ✨ Features

* 🌳 **Hierarchical Tree View**
  Navigate through **14 major AI branches** with up to **6 levels of depth**.

* 🔍 **Instant Search**
  Find any topic among **2100+ entries** with real-time filtering, automatic parent expansion, and match highlighting.

* 📘 **Smart Learning Prompts**
  Click **"ask ↗"** on any node to generate a structured, high-quality prompt for deep learning using AI tools.

* 🎨 **Responsive Design**
  Fully optimized for **mobile, tablet, and desktop**, with support for **dark/light themes**.

* ⚡ **High Performance**
  Built with **pure Vanilla JavaScript and CSS** — no dependencies, instant load time.
